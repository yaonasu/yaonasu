## 開発中・公開中のプロダクト

### プロジェクト管理/タスク管理ツール（ SeuPiM / ii-wf-taskMng ）

#### [SeuPiM](https://github.com/yaonasu/SeuPiM)（ Simple & Easy to Use Project & Issue Management Tool ）

Redmine のような課題管理ツールで  
個人や小規模チームでも扱いやすいシンプルなプロジェクト管理ツールです

* プロジェクト管理
* チケット管理
* 進捗管理
* サーバー構築不要（ファイルサーバー利用）


#### [ii-wf-taskMng](https://github.com/yaonasu/ii-wf-taskMng)

ショートカット・共有サーバを活用した回覧・ワークフローを実現するツールです  
タスク管理・ランチャーとしても活用頂けます


### エクセル差分抽出ツール（ ii-excel-diff / ii-exsh-diff / ii-exsh-diff-pro ）

エクセルを比較し、差分を抽出・可視化し、結果を記した帳票を作成するツールです  
３つそれぞれの特徴があり、シチュエーションに応じて 使い分けてご活用頂けます


#### [ii-excel-diff](https://github.com/yaonasu/ii-excel-diff)

２つのエクセルファイルを細かい設定なく簡単操作で比較・抽出できます


#### [ii-exsh-diff](https://github.com/yaonasu/ii-exsh-diff)

１つのエクセルファイル内の２つのシートに対して比較・抽出できます  
差分比較する列を指定することも可能です


#### [ii-exsh-diff-pro](https://github.com/yaonasu/ii-exsh-diff-pro)

ii-exsh-diff で複数ファイル、複数シートに対応したものとなります


### テキスト差分抽出ツール（ ii-win-merge / ii-diff / ii-text-diff ）

#### [ii-win-merge](https://github.com/yaonasu/ii-win-merge)

WinMerge の操作を肩代わり、差分結果をエクセルに出力するツールです


#### [ii-diff](https://github.com/yaonasu/ii-diff)

ii-win-merge とほぼ同等の機能で WinMerge なしで動作するツールです  
フォルダ指定で比較・差分抽出することができます


#### [ii-text-diff](https://github.com/yaonasu/ii-text-diff)

ii-diff のフリー版で ファイル指定で比較・差分抽出するツールです


### PDF ファイルの１枚１枚を画像化してエクセルに貼り付けるツール（ ii-pdf-paste2xlsx ）

#### [ii-pdf-paste2xlsx](https://github.com/yaonasu/ii-pdf-paste2xlsx)

PDF のページを１枚１枚画像にしたものを連続的に貼り付けたエクセルファイルを作成するツールです
